# 数字合成攻略器 - Android 版

## 怎么得到 APK 安装包？

你不需要安装任何开发工具。用 GitHub 免费自动编译：

### 第一步：上传代码到 GitHub（5 分钟）

1. 打开 https://github.com 注册账号
2. 点右上角 **+** → **New repository**
3. Repository name 填 `synthesis-helper`，点绿色的 **Create repository**
4. 点 **uploading an existing file** 链接
5. 把整个 `android` 文件夹里的所有文件和文件夹，**保持目录结构**拖进去上传
6. 点 **Commit changes**

### 第二步：自动构建 APK（3 分钟）

1. 在仓库页面点顶部 **Actions** 标签
2. 左边选 **构建 APK**
3. 右边点 **Run workflow** → 绿色 **Run workflow** 按钮
4. 等待约 3 分钟，出现绿色 ✅
5. 点进去 → 往下翻到 Artifacts → 点 **攻略器-APK** 下载
6. 解压 zip，得到 `app-release.apk`

### 第三步：安装到手机

把 APK 发到手机微信 → 点开安装即可。

> 手机需要 Android 8.0 以上。安装时如提示「未知来源」，点「允许」即可。

---

## 使用方法

1. 打开 App → 点 **开始攻略**
2. 授权悬浮窗 + 屏幕录制（两项都要点允许）
3. 切换到游戏 → 看到 🎯 红色气泡
4. **点一下气泡** → 开始实时监控
5. 气泡旁显示方向箭头（⬆️⬇️⬅️➡️），自动每 1.5 秒更新
6. 拖动气泡可移动位置
7. **再点气泡** → 暂停监控
